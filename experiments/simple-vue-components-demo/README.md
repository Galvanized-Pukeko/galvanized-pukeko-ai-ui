# Simple Vue Demo

This is a simple demo application showcasing the components from `@galvanized-pukeko/vue-ui` package.

## Components Demonstrated

- **PkButton** - Primary, secondary, and disabled button variants
- **PkInput** - Text and email input fields
- **PkInputCounter** - Number input with increment/decrement controls
- **PkCheckbox** - Checkbox input
- **PkRadio** - Radio button group
- **PkSelect** - Dropdown select
- **PkBarChart** - Bar chart visualization
- **PkPieChart** - Pie chart visualization
- **PkTable** - Data table
- **PkLogoLarge** - Large logo component

## Setup

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Usage

The app will be available at `http://localhost:5173` (or another port if 5173 is in use).

All form inputs are reactive and you can see the current state displayed at the bottom of the page.
