# @galvanized-pukeko/vue-ui

[![npm](https://img.shields.io/npm/v/@galvanized-pukeko/vue-ui)](https://www.npmjs.com/package/@galvanized-pukeko/vue-ui)

Vue.js component library for the [Galvanized Pukeko](../../README.md) framework. Provides a chat interface, dynamic form components, charts, and tables that LLM-powered agents can render on-the-fly via AG-UI and WebSocket protocols.
